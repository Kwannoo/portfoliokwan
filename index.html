projects

Here you can find my projects, and you can also download the project report by clicking on the logos!

MSc thesis: The stability of stablecoins: Are these coins stable or is the stability just a flip of the coin?


In collaboration with De Nederlandsche Bank, I worked on the project 'Plutus' which tries to predict the volatility of the stablecoins USDT and USDC, using time series prediction with both econometric and machine learning models. Additionally, some exploratory data analysis has been done with the aim of providing more explainability within the data and relations between the features.

The code was implemented in Python, and the analysis was conducted using libraries such as Scikit-learn, TensorFlow, and Pandas.

Supervisor: Dr Ronald Heijmans
Examiner: Dr Eric Pauwels
Grade: 8.0

BSc thesis: How to win in Checkers?


My bachelor's thesis focused on the game of checkers, specifically developing and implementing the game alongside the Minimax and Monte Carlo Tree Search (MCTS) algorithms. Both the game and the algorithms were entirely coded from scratch without relying on external packages. Additionally, a detailed comparison of the two algorithms' playstyles was conducted to determine which performs better.

The code was implemented in C++
Supervisor & Examiner: Dr Aske Plaat
Grade: 7.0
